ARQUIVOS E DIRETÓRIOS
^^^^^^^^^^^^^^^^^^^^^  

  bin/ - diretório destinado aos arquivos binários resultante da compilação. 
  bin/aplicativo.exe - arquivo executável deste software.
  -
  src/ - diretório destinado aos código-fontes.
  src/compilar.cmd - arquivo executável que realiza a compilação do projeto.
  src/executar.cmd - arquivo executável auxiliar que executa o software gerado.
  src/aplicativo.c - código-fonte único deste software.
  -
  ChangeLog - arquivo que documenta todas as alterações realizadas.
  LEIAME.txt - Este arquivo.
  ReleaseNotes - arquivo que descreve as principais características de cada versão lançada.
  Sugestoes.txt - arquivo que possui sugestões, acatadas ou não, para as próximas implementações.

COMPILAÇÃO
^^^^^^^^^^

  Para compilar este projeto, basta executar o arquivo "src/compilar.cmd".

  Pré-requisitos:
    - Possuir o GCC instalado.
    - Estar no PATH do sistema o diretório do executável "gcc.exe".
